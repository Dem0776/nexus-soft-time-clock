# 03 — Manual de usuario (colaborador)

Aplicación móvil **Nexus Soft Time Clock** para registrar tu asistencia.

## Iniciar sesión
1. Abre la app e introduce tu **correo** y **contraseña**. Si tu empresa lo requiere, indica el **código de empresa**.
2. La sesión se mantiene de forma segura en el dispositivo.

## Registrar asistencia
1. Pulsa **Registrar asistencia**.
2. La app obtiene tu **ubicación GPS** (concede el permiso si se solicita).
3. Escanea el **código QR** del centro de trabajo (o pega el token si se te indica).
4. Pulsa **Entrada** o **Salida** según corresponda.
5. Verás la confirmación. Si estás **sin conexión**, el registro se guarda y se **sincroniza automáticamente** al recuperar señal.

## ¿Por qué se rechazó mi registro?
El servidor valida cada registro. Motivos frecuentes:
- **Fuera del área permitida**: no estás dentro del radio del centro.
- **Precisión GPS insuficiente**: espera a tener mejor señal (a cielo abierto).
- **QR inválido o expirado**: pide al responsable el QR vigente.
- **Ubicación simulada / dispositivo no válido**: desactiva apps de localización falsa.

Un rechazo genera una **incidencia** que tu supervisor/RR.HH. puede revisar, y recibes una **notificación**.

## Pendientes de sincronizar
El icono de sincronización muestra cuántos registros están **pendientes**. Puedes forzar la sincronización pulsándolo. Ningún registro se pierde por falta de conexión.

## Consejos
- Mantén el **GPS activado** y con buena precisión.
- No uses aplicaciones de **ubicación falsa** ni dispositivos manipulados: los registros serán rechazados y marcados.
